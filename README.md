Random Quote Generator web application built with HTML, CSS, and JavaScript.

Users can:

Fetch random quotes from an API
Save favorite quotes in the browser
Delete quotes from favorites
Tweet quotes directly to Twitter

This project demonstrates front-end skills, API handling, DOM manipulation, event handling, and localStorage usage.

Features
✅ Display a random quote on page load
✅ Fetch quotes from Quotable API
✅ Fallback offline quotes in case the API fails
✅ Save quotes as favorites in the browser
✅ Delete quotes from the favorites list
✅ Tweet quotes directly

Technologies Used
HTML5 – Structure
CSS3 – Styling and layout
JavaScript (ES6) – API calls, DOM manipulation, event handling, localStorage
Quotable API – Fetch random quotes

Installation & Usage
Clone or download the repository
git clone https://github.com/YOUR-USERNAME/quote-generator-app.git
Open the project folder in VS Code or any code editor.
Open index.html in a browser
Recommended: use Live Server in VS Code for full functionality.
How to use the app:
Click New Quote to get a random quote.
Click Save Favorite to save it.
Delete quotes from the Favorites list if needed.
Click Tweet to share the quote on Twitter.
